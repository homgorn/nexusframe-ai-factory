```

```

- Что уже сделано (раздел "ЧТО УЖЕ СДЕЛАНО")
- Что нужно делать (раздел "СЛЕДУЮЩИЕ ШАГИ")
- Текущий checkpoint

### 3️⃣ В КОНЦЕ РАБОТЫ - СПРОСИ:
```

```

- Дату в MASTER_STATUS.md

## 📋 ЧТО ОБНОВЛЯТЬ В MASTER_STATUS.md

### Если сделали код:
```markdown

- [x] CIAN парсер работает  ← Отметь галочку
```

### Если нашли проблему:
```markdown
### Что НЕ работает (критично)
- [ ] Новая проблема: описание
```

### Если закончили checkpoint:
```markdown
#### Checkpoint 1.1: CIAN Парсер ✅ (100%)  ← Измени статус
- [x] Все задачи выполнены
```

### Обновить дату:
```markdown
**Дата последнего обновления**: 2025-01-13  ← Измени на текущую
**Активный чат**: Checkpoint 1.1 - CIAN Parser  ← Опиши что делали
```

